================================================================================
                         KEY DATA FLOWS
================================================================================

1. USER SCAN FLOW
   User → Select Scanner → Configure Options → Start Scan
   → Progress Bar Updates → Findings Detected → Results Stored (PostgreSQL)
   → Report Generated (HTML/PDF) → Download Available

2. PAYMENT FLOW
   User → Select Plan → Stripe Checkout → Payment (iDEAL/SEPA/Card)
   → Webhook (port 5001) → License Activated → Features Unlocked

3. ENTERPRISE EVENT FLOW
   Scanner → Finding Detected → Event Published (Redis)
   → Enterprise Orchestrator → Listeners Notified
   → Automated Actions (DSAR, Tickets, Alerts)

4. AUTHENTICATION FLOW
   User → Login Form → Bot Protection Check → Password Verification (bcrypt)
   → JWT Token Generated → Session Created (Redis) → Access Granted
