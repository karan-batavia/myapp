SECTION 4: INFRASTRUCTURE SECURITY
================================================================================

CHECK YOUR SERVER (45.81.35.202):

[ ] FIREWALL
    Run: sudo ufw status
    Should show:
    - 22/tcp ALLOW (SSH)
    - 80/tcp ALLOW (HTTP)
    - 443/tcp ALLOW (HTTPS)
    - Everything else DENIED

[ ] SSH SECURITY
    Run: sudo cat /etc/ssh/sshd_config | grep -E "PermitRootLogin|PasswordAuthentication"
    Should be:
    - PermitRootLogin no
    - PasswordAuthentication no (use SSH keys)

[ ] FAIL2BAN
    Run: sudo systemctl status fail2ban
    Should be: active (running)

[ ] SSL CERTIFICATE
    Run: sudo certbot certificates
    Check: Expiry date (auto-renew should be enabled)

[ ] OPEN PORTS
    Run: sudo netstat -tlnp
    Should ONLY show:
    - 0.0.0.0:22 (SSH)
    - 0.0.0.0:80 (Nginx)
    - 0.0.0.0:443 (Nginx)
    - 127.0.0.1:5000 (Streamlit - local only)
    - 127.0.0.1:5432 (PostgreSQL - local only)
    - 127.0.0.1:6379 (Redis - local only)

[ ] UPDATES
    Run: sudo apt update && sudo apt list --upgradable
    Keep system packages updated
