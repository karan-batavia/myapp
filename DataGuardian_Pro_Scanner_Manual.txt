====================================================
  DATAGUARDIAN PRO - QUICK SCANNER GUIDE
  How to Use Each Scanner & Get HTML Reports
====================================================

GENERAL: Log in > Sidebar > "New Scan" > Pick scanner > Set region > Scan > Download report

----------------------------------------------------
1. CODE SCANNER
----------------------------------------------------
- Pick "Code Scanner"
- Choose: Upload Files / Repository URL / Directory Path
- For repo: paste URL, set branch, add token if private
- Click "Start Code Scan"
- After scan: click "Download Report (HTML)"

----------------------------------------------------
2. DOCUMENT SCANNER
----------------------------------------------------
- Pick "Document Scanner"
- Upload files (PDF, DOCX, TXT, XLSX, CSV, RTF)
- Click "Start Document Scan"
- After scan: click "Download Report (HTML)"

----------------------------------------------------
3. IMAGE SCANNER
----------------------------------------------------
- Pick "Image Scanner"
- Upload images (PNG, JPG, TIFF, BMP, WEBP, GIF)
- Click "Start Image Scan"
- After scan: click "Download Report (HTML)"

----------------------------------------------------
4. DATABASE SCANNER
----------------------------------------------------
- Pick "Database Scanner"
- Select DB type (PostgreSQL/MySQL/SQL Server/Oracle/MongoDB)
- Fill in: Host, Port, Database, Username, Password
- Click "Test Connection" first, then "Start Database Scan"
- After scan: click "Download Report (HTML)"

----------------------------------------------------
5. WEBSITE SCANNER
----------------------------------------------------
- Pick "Website Scanner"
- Enter website URL (e.g. https://example.com)
- Choose scan depth: Light (1 page) / Standard (5) / Deep (25)
- Click "Start Website Scan"
- After scan: click "Download Full Report (HTML)"

----------------------------------------------------
6. AI MODEL SCANNER (EU AI Act)
----------------------------------------------------
- Pick "AI Model Scanner"
- Follow the 4-step wizard:
  Step 1: Enter system name, purpose, use case
  Step 2: Set data scope and oversight
  Step 3: Check compliance requirements
  Step 4: Enter company info for penalty calc
- Report generates automatically at the end

----------------------------------------------------
7. DPIA SCANNER
----------------------------------------------------
- Pick "DPIA Scanner"
- Follow the 5-step wizard:
  Step 1: Project name, controller, purpose
  Step 2: Check data types (personal, sensitive, BSN, etc.)
  Step 3: Check risk factors (large-scale, automated, etc.)
  Step 4: Check safeguards (encryption, access controls, etc.)
  Step 5: Review summary, click "Generate DPIA Report"

----------------------------------------------------
8. SOC2 SCANNER
----------------------------------------------------
- Pick "SOC2 Scanner"
- Enter repository URL + optional access token
- Select report type (Type I or Type II)
- Check trust criteria (Security, Availability, etc.)
- Click "Start SOC2 Assessment"
- After scan: report shown on screen

----------------------------------------------------
9. SUSTAINABILITY SCANNER
----------------------------------------------------
- Pick "Sustainability Scanner"
- Choose: Repository URL / Cloud Provider / Upload Config
- Click "Start Sustainability Analysis"
- After scan: results shown on screen

----------------------------------------------------
10. AUDIO/VIDEO DEEPFAKE SCANNER
----------------------------------------------------
- Pick "Audio/Video Scanner"
- Choose: Audio (MP3/WAV/FLAC/M4A) or Video (MP4/AVI/MOV/MKV)
- Upload one file
- Click "Start Deepfake Analysis"
- After scan: authenticity score + risk level shown

----------------------------------------------------
11. DATA SOVEREIGNTY SCANNER (Enterprise Only)
----------------------------------------------------
- Pick "Data Sovereignty Scanner"
- Choose mode:
  a) Upload Config (.tf, .json, .yaml, .bicep)
  b) GitHub Repo URL + branch + scan depth
  c) Manual Input (add data locations + flows)
- Click "Analyze Sovereignty"
- After scan: click "Download HTML Report"

----------------------------------------------------
12. ADVANCED AI SCANNER
----------------------------------------------------
- Pick "Advanced AI Scanner"
- Choose analysis type (PII / Contract / Policy / Consent)
- Provide input: Upload doc / Paste text / Enter URL
- Click "Start Advanced AI Analysis"
- After scan: compliance score + insights shown

----------------------------------------------------
13. ENTERPRISE CONNECTORS
----------------------------------------------------
- Pick "Enterprise Connector"
- Select connector:

  Microsoft 365: Connect via OAuth > scan OneDrive/SharePoint/Outlook
  Google Workspace: Connect via OAuth > scan Drive/Gmail/Docs/Sheets
  Exact Online: Connect via OAuth > scan customers/suppliers/employees
  Salesforce: Enter instance URL > connect > scan contacts/leads/accounts
  SAP: Select system type > enter host/credentials > scan master data

- Click the "Start [Connector] Scan" button
- After scan: findings shown on screen

====================================================
NOTE: HTML report downloads require a paid plan.
Free trial users can view results on screen only.
