## 🚨 Don't See an App? Create One

**If the table says "No records to display":**

1. Look for **[New Connected App]** button at top of page
2. Click it
3. Fill in:
   ```
   Connected App Name: DataGuardian Pro
   API Name: DataGuardian_Pro
   Contact Email: [your email]
   
   ☑ Enable OAuth Settings (CHECK THIS BOX)
      Callback URL: http://localhost:5000
   ```
4. Click [Save]
5. After save, your Consumer Key & Secret appear on the page
6. Copy them as shown in Steps 3-4 above
